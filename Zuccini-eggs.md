### 1. Підготовка інгредієнтів

_Приготуйте все заздалегідь, щоб у кадрі було естетично._

- **Цукіні (Zucchini):** 2 середніх (близько 500-600г / 1.2 lbs).
- **помідорів чері:** 5-6 шт. (для кольору та свіжості).
- **Каламата оливки:** 5-6 шт. (для глибини смаку).
- **Фета (Feta Cheese):** 150-200г (5.3 - 7 oz) — _справжня грецька, з овечого молока_.
- **Зелена цибуля (Green Onions):** 2-3 пера.
- **Свіжі трави:** Гілочки орегано та м’яти (з вашого балкона).
- **Оливкова олія (EVOO):** 1 ст. л.
- **Чорний перець:** за смаком.
- **Важливо:** Сіль додаємо тільки до цукіні для виведення вологи, у суміш не додаємо, бо фета солона.

### 2. Підготовка інструментів

- Тертка (велика).
- Велика миска для змішування.
- Чистий кухонний рушник (або марля) — _це буде ключовий кадр для «хаку» з витисканням вологи_.
- Форма для запікання, що підходить для вашого Air Fryer (силіконова або металева діаметром 18-20 см).
- Вінчик або виделка.
- Пензлик для олії.

---

### 3. Покроковий процес (Action Plan)

1.  **Підготовка:** Натерти цукіні в миску, посолити, через 5 хв відтиснути через рушник.
2.  **Процес у формі:**
    - Змастіть форму для запікання оливковою олією.
    - Покласти відтиснуті цукіні у силіконову форму.
    - Розбити 5 яєць.
    - Додати нарізану зелену цибулю, свіжий орегано/м'яту.
    - Розкришити фету.
    - Додати дрібку перцю та оливкову олію.
    - **Перемішати все прямо у формі.**
3.  **Декор (краса для кадру):**
    - Вставити кілька половинок чері зрізом догори.
    - Додати пару оливок Каламата.
    - Зверху можна трохи збризнути оливковою олією для золотистої скоринки.
    - Посипати дрібкою сухого орегано.
4.  **Запікання:** В аерогриль на 180°C (350°F) **20-25 хвилин**. Поверхня має стати золотисто-коричневою, а центр — пружним (не рідким).

### Фінальний дотик

15. Дістаньте форму. Дайте страві «відпочити» 5-10 хвилин перед нарізанням (це важливо для стабілізації структури).
16. Наріжте на трикутники (як пиріг).
17. Подавайте з гілочкою свіжого орегано зверху.

### Порада для подачі (Сервірування):

Подавайте цей пиріг на дерев’яній дошці (можна на тій самій Teak Board, яку ви нещодавно оглядали). Поруч покладіть жменю свіжого листя салату (Maroulosalata) або просто пару скибочок лимона. Лимонний сік на яйця з цукіні — це дуже по-грецьки і додає свіжості.
